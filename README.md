# 🎓 Student Career & Internship Tracker

A full-featured internship and career management platform built to help students organize their internship journey, track learning progress, and monitor recruitment activities in one place.

The application provides a clean and responsive user experience with internship application tracking, analytics dashboards, learning goal management, authentication, and advanced filtering capabilities. It is designed with modern frontend development practices using React, TypeScript, Redux Toolkit, and Tailwind CSS.

## 🚀 Features

* Secure mock authentication with session persistence
* Protected routes using React Router
* Internship application management (Create, Read, Update, Delete)
* Search, filter, and sort applications by company, status, and location
* Interactive analytics dashboard with charts and recruitment insights
* Learning progress tracker for courses, certifications, and skill development
* Dark mode support
* Responsive design for mobile, tablet, and desktop devices
* Toast notifications and confirmation dialogs
* Error handling with React Error Boundaries
* Pagination for better data management
* Mock API integration using DummyJSON
